# African (Mobile)

Minimal Expo (React Native) starter for the African mobile app.

## Prerequisites
- Node.js (16+)
- npm or yarn
- (Optional) Expo CLI: `npm install -g expo-cli` or use `npx expo`

## Run locally
1. Install dependencies:
   npm install
2. Start the Expo dev tools:
   npm run start
3. Open on a device:
   - Scan the QR code with Expo Go (Android/iOS)
   - Or run on emulator:
     npm run android
     npm run ios

## Build / Publish
- Web: `npm run build:web`
- For native builds and publishing, see Expo docs: https://docs.expo.dev/

## Notes
- Add image assets referenced in app.json to /assets (icon.png, splash.png, adaptive-icon.png, favicon.png) or update paths.
